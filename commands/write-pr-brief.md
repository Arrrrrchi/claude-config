人間レビュアー向けの PR 要約（Human PR Review Brief）を作成してください。

- `pr-review-brief` スキルのテンプレに沿って、実装内容・受け入れ条件との対応・重点確認点・触った重要ファイル・リスク領域・実行済みチェック・レビュー結果・残リスクをまとめる。
- 埋められない欄は「該当なし/未実施」と明示する。
- 出力は PR 本文に置くのが既定（プロジェクトの慣習があればそれに従う）。
- **PR の作成自体はユーザー承認後**。このコマンドは要約の生成まで。
