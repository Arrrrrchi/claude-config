---
description: AI の失敗・見落とし・誤判断を記録し、再発防止策の昇格先を分類
---

AI（自分）が起こした失敗・見落とし・誤判断を記録してください。

- `failure-log` スキルのフォーマットに沿って、Failure / Impact / Cause / Detection / Prevention を記録する。
- 再発防止策をどこに昇格すべきか（rules / hooks / agents / skills / CI / tests / CLAUDE.md）を `Should Update` で分類する。
- 実際の失敗事例はプロジェクト側に保存する（グローバルにはフォーマットのみ）。
- **同じ失敗が再発している**なら、`quality-learner` スキルで各ファイルへ実反映するところまで進める。
